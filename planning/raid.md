# raid.md — RAID Log

Risks, Assumptions, Issues, and Decisions for this project. Jarvis-owned. Append-only — rows are updated in status but never deleted.

ID convention: monotonic, zero-padded, never reused.
- **R-NNN** — Risks
- **A-NNN** — Assumptions
- **I-NNN** — Issues
- **D-NNN** — Decisions *(separate ID space from PLAN.md Decisions Ledger — these are PM-level decisions; PLAN.md tracks all planning decisions)*

Confidence labels apply to every row: `actual` | `estimate` | `extrapolation`. Sources must be named in Notes when label is `actual`.
